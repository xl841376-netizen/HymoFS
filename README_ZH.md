# HymoFS

> **警告**：这是一个实验性的内核修改项目。

HymoFS 是一个内核补丁，旨在通过修改 `getdents` 系统调用将虚拟条目注入到目录列表中。

## 风险与限制

*   **系统不稳定性**：修改核心文件系统系统调用具有高度侵入性，可能导致内核恐慌 (Kernel Panic) 或系统死机。
*   **性能影响**：注入逻辑会增加每次目录读取操作的开销，可能会降低 I/O 性能。
*   **兼容性问题**：此补丁可能会与其他文件系统修改或安全模块发生冲突。
*   **数据完整性**：虽然旨在进行只读注入，但实现中的错误理论上可能会影响文件系统的稳定性。

## 使用说明

此补丁仅供了解内核级修改风险的开发者和高级用户使用。
